**重要度**: 高

**指摘内容**:
現在、GitHub APIクライアントはトークンをパラメータとして受け取る設計だが、
next-authとの統合が未完了。

**現状**:
```typescript
// github-client.ts
interface GitHubClientConfig {
  owner: string;
  repo: string;
  token: string;  // どこから取得？
}
```

**推奨対応**:
1. next-auth設定完了
2. セッションからトークン取得
3. 環境変数 `GITHUB_OWNER`, `GITHUB_REPO` 設定

**対応予定**: Phase 5
